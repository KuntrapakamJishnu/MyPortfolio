Project Overview

Doc Query AI is an advanced document analysis platform that revolutionizes how users interact with their documents. Using Retrieval-Augmented Generation (RAG), the system enables users to ask natural language questions and receive precise, contextually relevant answers extracted from their documents.

The project leverages semantic search technology to find the most relevant document sections and combines them with AI language models to generate accurate responses. Built with Python FastAPI backend, Streamlit interface, and React.js frontend, the platform demonstrates exceptional performance in document understanding. It was selected as a finalist in the prestigious Infosys Bangalore DC Program among 30+ competing teams, showcasing its innovation and technical excellence.
