🔗 Live Demo 💻 Source Code

Project Overview

OneDrop is a cross-platform emergency blood donation application that revolutionizes how donors and recipients connect during critical situations. The platform leverages real-time geolocation APIs to match donors with nearby emergency requests, minimizing response latency and potentially saving lives.

The system features a comprehensive management interface for donors, hospitals, and emergency requests. It integrates SMS and email authentication for secure donor verification, implements a robust notification system to alert potential donors instantly, and provides real-time tracking and navigation. The application won a Health Hackathon, demonstrating its practical impact and innovative approach to emergency blood donation.
